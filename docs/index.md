# Manual de Prácticas de IoT - Particle Photon 2

**Bienvenido al manual interactivo más completo de la materia IoT**

Aquí encontrarás **todas las prácticas** que realizamos durante el semestre, con simuladores en vivo, códigos listos para copiar, diagramas y evidencias reales.

---

##  ¿Cómo usar este manual?

- Usa el menú de la izquierda para navegar entre prácticas
- Cada práctica tiene **simulador Wokwi** para probar antes de armar el circuito
- Puedes copiar el código con un solo clic
- Marca las prácticas como completadas
- Modo oscuro automático
